Static Keyword Easy Explanation
Jab aap static use karte ho class ke property ya method ke saath, 
iska matlab hai
Ye property ya method class ka part hai, instance (object) ka nahi.
Aapko object banaye bina bhi class ke static property/method ko 
use kar sakte ho.
Agar aap normal property/method use karte ho
Har object banane pe alag memory allocate hoti hai.
Agar aap static property/method use karte ho
Sirf ek hi copy memory me store hoti hai → memory efficient.
